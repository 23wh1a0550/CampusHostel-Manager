## 📌 Project Overview
The **Campus Hostel Management System** is a full-stack web application designed to manage hostel operations efficiently within a campus. It enables administrators and students to interact with the system for managing hostels, bookings, and analytics.

---

## 🎯 Objectives

- Simplify hostel management  
- Maintain accurate hostel records  
- Enable student registration & booking  
- Provide analytics for decision-making  

---

## 🛠️ Tech Stack

### 💻 Frontend
- React.js  
- Axios  
- React Router  

### ⚙️ Backend
- Node.js  
- Express.js  

### 🗄️ Database
- MongoDB  
- Mongoose  

---

## ✨ Features

### 👤 User Management
- User Registration with validation  
- Role-based access (Admin / Student)  
- Secure Login System  

### 🏠 Hostel Management
- Add / Update / Delete hostels  
- Track bed availability  
- Manage hostel types  

### 📅 Booking System
- Hostel booking for students  
- Booking status tracking  
- View booking details  

### 📊 Analytics
- Total hostels & bookings  
- Bed availability  
- Most popular hostel  
- Booking trends  

### 🔄 CRUD Operations
- Full CRUD for users & hostels  
- RESTful API implementation  

### 🔍 Advanced Queries
- Filtering (e.g., AC hostels)  
- Sorting & limiting data  
- Efficient retrieval

## 🔗 API Endpoints

| Method | Endpoint        | Description         |
|--------|----------------|---------------------|
| GET    | /hostels       | Get all hostels     |
| POST   | /hostels       | Add hostel          |
| PUT    | /hostels/:id   | Update hostel       |
| DELETE | /hostels/:id   | Delete hostel       |
| POST   | /login         | Admin login         |

## 🚀 Installation & Setup

### 🔧 Prerequisites
- Node.js  
- MongoDB  
- npm / yarn  

### ⚡ Steps

```bash
# Clone repository
git clone https://github.com/your-username/campus-hostel-management.git

# Backend setup
cd backend
npm install
node server.js

# Frontend setup
cd frontend
npm install
npm start
